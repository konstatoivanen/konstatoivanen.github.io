GLOBAL ILLUMINATION

> Rasterize scene geometry into a RLE height field
   for mapping playable area.
> Bake sh9 probes for each height field span & minimize
   probe count by sharing similar probes between spans.
> Bilinearly interpolate between sampled cells.

© Critical Force Ltd. Used with permission.

PBR SKIN TOOLS

> Projection map textures onto models.
> Render layers into albedo, normal map or maor targets.
> Export skin to texture files or generate at runtime.
> Save in memory by sharing resources between skins.

© Critical Force Ltd. Used with permission.

MOBILE PBR

> Single directional light support.
> 3 hdr octahedron environment maps for reflections.
> Spherical harmonics for indirect lighting.
> Directional light extracted from sh9 values.
> OEM sample interpolation between volumes.

© Critical Force Ltd. Used with permission.

VOLUMETRIC LIGHTING

> 4 samples per texel.
> Wanghash noise for dithering.
> Half resolution rendering.
> Bilateral upsampling.
> Frame blending to resolve noise.
> Gaussian 5x5 PCF for shadows.
> D-lambert to emulate in scattering.

2D POLYGON FRAGMENTATION

A Procedural 2D mesh pattern
generator. Patterns can be edited,
saved, created manually or clipped
by a convex polygon.
Pattern fragments can then be
materialized into physics objects.

GENERIC IK SOLVER

A multithreaded IK solver
using CCD for n > 2 chains
and simple trigonometry
for n < 2 chains.
