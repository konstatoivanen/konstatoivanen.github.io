VOLUMETRIC LIGHTING

Properties:
- 4 samples per texel.
- Wanghash noise for dithering.
- Half resolution rendering.
- Bilateral upsampling.
- Frame blending to resolve noise.
- Gaussian 5x5 PCF for shadows.
- D-lambert to emulate in scattering.

SIDE SLASHER

Our goal was to create a vertical slice of a 2D
melee action game for a technology
workshop course. This we achieved for the
most part (a few novelty features missing).

TEXT & DRIVE

A game about the dangers of
texting & driving at the same time.

BOID SIMULATION

A DirectX compute boid simulation
with drawable physics geometry.

TANK GAME

A 1v1 local multiplayer tank game.
Made using OpenTK.
Properties:
- Voronoi based level generation.
- 2D physics solver.

2D POLYGON FRAGMENTATION

A Procedural 2D mesh pattern
generator. Patterns can be edited,
saved, created manually or clipped
by a convex polygon.
Pattern fragments can then be
materialized into physics objects.

GENERIC IK SOLVER

A multithreaded IK solver
using CCD for n > 2 chains
and simple trigonometry
for n < 2 chains.

HIGHLIGHTING SYSTEM

Renders a dynamic stack of objects
into a buffer to create a highlighting
post effect. Supports individual
properties such as, fade in/out,
color and thickness.

PLAYER CONTROLLER

Properties:
- Step offset
- Cylinder collision
- Vaulting
- Wallrunning
- Sliding
- Spring smoothing,
- Stick to moving platforms

ROPE SIMULATION

An easy to tweak but physically
incorrect simulation of
interconnected springs.
Rendered with OpenTK
(a C# wrapper for OpenGL).

AI SYSTEM

A versatile framework that allows
for easy derivations from baseclass
and the creation of almost any kind
of dynamic behavior such as flocking,
flanking, searching or apprehension.
Stimuli can be fed through visual,
auditory or olfactory means.

BIPED CHARACTER CONTROLLER

A Robust biped character controller with
support for features such as, physics
fade in/out,
apply animation movement to physics,
strafing, scripted sequences, foot IK
and weapon to target IK.

INSTANCE MANAGER

Object pooling system with extended
features such as, hashed object ids,
impact fx from physic material,
positional audio cues and
procedural mesh caching.
