A tool for converting cylindrical IBL textures to GGX convolved octahedron mapped IBL textures.
Output is convolved onto mip map layers using monte carlo integration & GGX importance sampling based on user defined roughness values.
The tool can write the result in either RGBM, RGB9E5 or RGBA16F format.
The resulting image can then be used as an IBL background in other rendering software.
