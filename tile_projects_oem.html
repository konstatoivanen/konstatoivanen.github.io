Hobby project // An Offline GGX reflection probe convolver:
> Applies GGX Monte carlo integration to cylindrical IBL textures.
> Outputs octahedron mapped IBL textures.
> Different roughness convolutions stored in output mip-levels.
> Number of levels and level roughnesses defined through CLI args.
> Outputs a KTX texture in either RGBM, RGB9E5 or RGBA16F format.
> Resulting texture can be used by other rendering software for lightweight IBL.
